# Clod Desktop

Electron macOS shell generated by Clod.

## Development

```bash
cd /Users/yh/Desktop/clod2/clod-agent/desktop-app
npm install
npm start
```

## Build

```bash
bash scripts/setup-macos-app.sh
```

The generated app is expected at:

```text
/Users/yh/Desktop/clod2/clod-agent/desktop-app/dist/mac-arm64/Clod.app
```

Backend command:

```text
python3 {project_dir}/api.py
```

Start URL:

```text
http://localhost:{port}
```
